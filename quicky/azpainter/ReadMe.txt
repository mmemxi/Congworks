==== 概要 ====

使いやすさを重視し、シンプルにまとめた軽量なペイントソフトです。
手軽に絵を描くことを目的としています。


==== 動作環境 ====

Windows98 以降
（Win7 64bitでの動作報告あり）

※MMX対応CPU必須
（1997年頃以降のパソコンであれば大丈夫です）


==== インストール ====

アーカイブを適当なフォルダに解凍してください。


==== アンインストール ====

ファイルを削除するだけです。
レジストリは使用していません。


==== バージョンアップ ====

各ファイルを上書きコピーしてください。
ただし、palette.dat・texture.apt については
現在のパレットやテクスチャを維持したい場合にはコピーしないでください。


==== ファイル構成 ====

[AzPainter2.exe]  実行ファイル
[palette.dat]     デフォルトパレットファイル
[texture.apt]     デフォルトテクスチャライブラリファイル


==== その他 ====

PNGなどのZIP圧縮/展開部分に以下のライブラリを使用しています。
zlib ver 1.2.5 Copyright (C) 1995-2010 Jean-loup Gailly and Mark Adler

このソフトはフリーソフトです。
著作権は作者であるAzelが所持しています。

使用、転載、配布に制限はありません。
雑誌等への収録も自由です。
転載、配布時はアーカイブファイル名および格納ファイルを改変することを禁止します。

このソフトを使用を使用して発生したいかなる損害にも作者は責任を負わないものとします。
