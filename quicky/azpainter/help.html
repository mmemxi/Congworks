▼ メニュー

＜ファイル＞

  • 固定フォルダと形式で保存
    「固定フォルダと形式の設定」 で指定されたフォルダと形式で保存します。
  • 固定フォルダと形式の設定
    「固定フォルダと形式で保存」 時の保存フォルダと形式を指定します。

＜編集＞

  • 現在のレイヤをコピー（24BIT・アルファ無）
    現在のレイヤのイメージを、白色の背景と合成した状態で24BITイメージでクリップボードにコピーします。
  • 現在のレイヤをコピー（32BIT・アルファ有）
    現在のレイヤのイメージを、32BITイメージでアルファ値を維持した状態でクリップボードにコピーします。
    ※これでコピーした場合、32BITイメージとして貼り付けられるソフトで貼り付けてください。
  • 全体のイメージをコピー
    全レイヤを合成した状態の24BITイメージをクリップボードにコピーします。
  • 新規レイヤに貼り付け（通常）
    クリップボードのイメージを、アルファ値を考慮せず新規レイヤに貼り付けます。
  • 新規レイヤに貼り付け（アルファ有効）
    クリップボードのイメージを、アルファ値を考慮して新規レイヤに貼り付けます。
    ※クリップボードのイメージが32BITでないと実行できません。
    ※確実にアルファ値が有効な状態でコピーされたイメージであることがわかっている時に使ってください
    （32BITイメージであってもアルファ値がすべて0であったり適当な値である場合があります）
  • 全体を〜
    全てのレイヤを対象に操作を行います。
  • イメージサイズ変更
    イメージのサイズを変更します（サイズの拡張や切り取り）
  • イメージサイズ拡大縮小
    全てのレイヤを指定サイズに拡大縮小することでイメージサイズを変更します。

＜選択範囲＞

※選択範囲のコマンドは、（一部のコマンドを除き）選択範囲が存在しない場合は全体が選択されているものと見なして処理されます。

  • 座標指定選択
    選択範囲の座標を直接値で指定します。
  • コピー
    範囲内のイメージを内部データにコピーします。
    ※クリップボードにはコピーされません。
  • 切り取り
    範囲内のイメージをコピーした後、範囲内を透明にします。
  • 貼り付け
    「コピー」「切り取り」で内部データにコピーされたイメージを貼り付けます。
  • 塗りつぶし
    範囲内を現在の描画色（濃度：255）で塗りつぶします。テクスチャなども有効です。
  • 消去
    範囲内を濃度255の消しゴムで消去します。テクスチャなども有効です。
  • ［他へ出力］クリップボード（24BIT・アルファ無）
    範囲内のイメージを、白色の背景と合成させた状態で24BITイメージとしてクリップボードにコピーします。
  • ［他へ出力］クリップボード（32BIT・アルファ有）
    範囲内のイメージを、32BITイメージでアルファ値を維持した状態でクリップボードにコピーします。
  • ［他へ出力］アルファ付きPNGファイル
    範囲内のイメージを、32BITイメージでアルファ値を維持した状態でPNGファイルに保存します。
  • ［他から貼り付け］クリップボード（通常）
    クリップボードのイメージをアルファ値を考慮しない状態で貼り付けます。
  • ［他から貼り付け］クリップボード（アルファ有効）
    クリップボードの32BITイメージをアルファ値を考慮した状態で貼り付けます。
  • ［他から貼り付け］ファイル
    ファイルからイメージを読み込んで貼り付けます。
  • ［イメージを並べる］
    範囲内のイメージを、全体・横方向・縦方向に端まで並べます。
  • 合成して貼り付け
    「貼り付け」でイメージを貼り付けるとき、現在のイメージと貼り付けるイメージを合成させます。
    チェックがOFFの場合は、貼り付けるイメージがそのまま上書きコピーされます。

＜表示＞

  • 全ウィンドウを閉じる/復元
    現在表示されているウィンドウを全て閉じます。
    全て閉じられた状態で実行すると、以前の状態に復元されます。
  • 全画面表示
    デスクトップ全体に表示します。
  • 背景をチェック柄で表示
    イメージの背景をチェック柄で表示します。
  • キャンバス左右反転表示
    キャンバスのイメージを左右反転して表示します。
    左右反転されるのは表示上だけなので、実際のイメージは反転しません。
  • グリッド表示
    グリッドを表示します。
  • グリッド設定
    [イメージ中央十字線] … 通常のグリッド線と共にイメージの中心にあたる部分に十字線を表示します。
    [1pxグリッド表示条件] … 表示倍率が指定された倍率以上の時は1pxごとにグリッドを表示します。1pxグリッドは、グリッドを非表示にしていても有効です。

▼ キャンバス

  • 表示位置移動
    中ボタン、もしくはスペースキーを押しながらドラッグすると、キャンバス上のイメージの位置を移動できます。
  • 表示倍率の拡大縮小
    マウスホイールを上へ移動すると1段階拡大、下へ移動すると1段階縮小できます。
  • スポイト
    （ツールが選択範囲の時を除き）イメージ上で右クリックすると、その位置の色を取得して現在の描画色にセットできます。
    ＋Ctrlキー / カレントレイヤ上の色を取得します。
    ＋Shiftキー / 取得した色をマスク色の1番目にセットします。
    ＋Altキー / 取得した色をマスク色に追加します。
  • 中間色作成
    Backspaceキーを押しながらイメージ上で2回、2つの色の上を左クリックすると、その2色の中間色を現在の描画色にセットできます。
    （1回目のBSキー＋左クリックで1つ目の色を取得、1回目のBSキー＋左クリックで2つ目の色を取得した後、中間色が作成されます）
    ＋Shiftキー / 2色の間に、中間色を2つ作成して描画色と背景色にそれぞれセットします。
  • 色の置き換え
    Enterキーを押しながらイメージ上を左クリックすると、色の置き換えが出来ます。
    押された位置のカレントレイヤ上の色を取得し、カレントレイヤ全体でその色を描画色や透明色に置き換えます。
    （選択範囲がある場合は、その範囲内だけ処理します）

    クリックした位置の色が透明でない / その色を描画色に置き換えます。
    クリックした位置の色が透明 / 透明色を描画色に置き換えます。
    ＋Shiftでクリックした位置の色が透明でない / その色を透明色に置き換えます。
  • レイヤイメージ1px移動
    Ctrl＋矢印キーでカレントレイヤのイメージを上下左右に1px移動できます。
    （※移動ツールで1px移動するのと同じなので、移動後に消えた部分のイメージは位置を戻しても戻りません）

▼ ツールウィンドウ

■ ＜ツール＞

  • 鉛筆
    ドット編集用です。※筆圧無効

    [形状] … 形状を指定します。
    [重ね塗り] … 重ね塗りをするかを指定します。
  • 1px細線
    1px固定サイズで、自由線を描画した時に線が細くなるように処理されます。※筆圧無効
  • ペン
    主線用のアンチエイリアス付きのペンです。

    [筆圧S] … ブラシサイズで筆圧を有効にします。
    [筆圧A] … 濃度で筆圧を有効にします。
    [アンチエイリアス] … 線のアンチエイリアス処理を行うか。
    [曲線] … 線を曲線で補間します。
    [形状/柔] … 線が柔らかくなるようにブラシの形状を使います。
    [補正] … 手ぶれ補正の強さを指定します。
  • ペン2
    色塗り用のペンです。

    [最小サイズ] … 筆圧感知時の最小ブラシサイズを％単位で指定します。
    [最小濃度] … 筆圧感知時の最小濃度を％単位で指定します。
  • ブラシ
  • 水彩
    [水の量] … A〜Dまで指定でき、Aが一番水の量が少なく、Dが一番水の量が多くなっています。
  • スプレー
  • 消しゴム（ドット）
    ドット編集用の消しゴムです。※筆圧無効
  • 消しゴム（ペン）
    ペンでの消しゴムです。
  • 消しゴム（ブラシ）
    ブラシ形状での消しゴムです。
  • ぼかし
    描画した部分をぼかします。※筆圧無効
    濃度の値が大きいほどぼかしの強度は強くなります。
  • 指先
    指先でなぞったように描画した部分を延ばします。※筆圧無効
    濃度の値が大きいほど強くなります。
  • 覆い焼き
    描画した部分の色を明るくします。
  • 焼き込み
    描画した部分の色を暗くします。

  • 塗りつぶし
    左クリックした位置から同じ色の範囲を塗りつぶします。

    《 判定元レイヤについて 》
    塗りつぶし時には、塗りつぶす範囲を判定するレイヤと実際に塗りつぶすレイヤを別にすることができます。
    各レイヤで「塗潰判定元」というチェックがありますが、判定元にしたいレイヤでこのチェックをONにして塗りつぶしを行うと、塗りつぶす範囲の判定はそのレイヤで行い、現在のレイヤでその範囲を塗りつぶします。
    これを使うと、線画のレイヤを判定元にして色塗りのレイヤで塗りつぶして、線画で囲まれた範囲を別のレイヤに塗りつぶすことができます。
    ※判定元に指定されているレイヤが存在しない場合は、通常通り現在のレイヤで塗りつぶします。
    ※判定元には複数のレイヤを指定することができますが、判定元に2つ以上のレイヤを指定した場合は、判定用の色として各レイヤのRGB値は合成せずアルファ値のみ合成した色を使います。複数指定の場合は透明部分を塗りつぶすことを想定しているので、複数指定で透明部分以外を塗りつぶす場合は正しく判定されません。

    [判定元レイヤ有効]
    レイヤの 「塗潰判定元」 を有効にして塗りつぶします。
    OFFの場合は、「塗潰判定元」 に指定されているレイヤが存在しても常にカレントレイヤを対象にします。
    [境界自動判定]
    透明部分を塗りつぶす時のみ有効です。
    ONの場合は、色ではなくアルファ値から境界を自動的に判定して、ペンなどで描画されたアンチエイリアス部分も綺麗に塗りつぶします。
    ただし、細長い部分などは完全に塗りつぶせない場合もあります。
    OFFの場合は、通常通り完全に透明な範囲だけを塗りつぶします。
    [消しゴム]
    同じ色の範囲または不透明範囲を消しゴムで消去して透明にします。
    [↑不透明範囲]
    「消しゴム」 がONの場合に、同色範囲ではなく不透明な範囲（アルファ値が0でない範囲）を消去します。
    ペンやブラシなどで描いた線を綺麗に消すことができます。

    (キー操作)
    ＋Ctrlキー / 消しゴムで塗りつぶします。
    ＋Ctrl＋Shiftキー / 不透明範囲を消しゴムで塗りつぶします。
    ＋Shiftキー / 一時的に 「境界自動判定」 のチェックを反転して塗りつぶします。
    ＋Altキー / 一時的に「判定元レイヤ有効」のチェックを反転して塗りつぶします。
    （反転→チェックがONの場合は一時的にOFFとして扱い、OFFの場合は一時的にONとして扱います）

  • グラデーション
    左ドラッグで直線を指定し、グラデーションを描画します。
    グラデーションの形は描画タイプの選択で変更できます。
    （自由線・直線→線形、 四角枠・四角塗りつぶし→矩形、 円枠・円塗りつぶし→円形、 集中線→放射状）

    [タイプ]
    グラデーションの色を指定します。
    「効果1」は円グラデーション用です。
    [反転]
    色の左右を反転します。
    「白→黒」であれば「黒→白」にします。
    [階調]
    「1/*」の＊の値が大きいほど色数が減ります。

    ＋Ctrlキー / 直線指定時に押されていると45度単位の線にします。
    ＋Shiftキー / 最初に直線で指定した時のその範囲内のみ描画します。
    ＋Altキー / 通常、端の部分は端の色で塗りつぶされますが、その範囲でも色を繰り返して描画します。

  • テキスト
    テキストを描画します。
    描画したい部分を左クリックするとテキストなどを指定するウィンドウが出るので、それぞれ指定した後「描画」ボタンで描画してください。
    また、ウィンドウが出ている間にキャンバス上を左クリックすると描画位置を変更できます。
    テキストエディット内にカーソルがある時にAlt＋矢印キーで描画位置を1px上下左右に移動できます。

    [フォント名]
    先頭に「＠」が付いているフォントは縦書き用です。
    「縦書き」のチェックと併せて使ってください。
    [サイズ] … 文字のサイズをpt単位で指定します。
    [字間] … 文字と文字の間の間隔をpx単位で指定します（負の値も可）
    [行間] … 行と行の間の間隔をpx単位で指定します（負の値も可）

  • 移動
    現在のレイヤのイメージを移動させます。
    ＋Shiftキー / 水平方向に移動させます。
    ＋Ctrlキー / 垂直方向に移動させます。

  • 選択範囲
    左ドラッグで範囲を指定し、その範囲で各操作を行います。
    右クリックすると選択範囲のメニューが表示されます。

    範囲をドラッグすると、その範囲のイメージを移動できます。
    イメージを確定させる時は範囲外をクリックするなどして選択を解除してください。
    また、最初にドラッグする時にAltキーを押しながらドラッグさせると、移動ではなくコピーとなり元の範囲のイメージは消去されません。
    ドラッグ移動時は、＋Shiftキーで水平移動、＋Ctrlキーで垂直移動ができます。


■ ＜描画タイプ＞

  • 自由線
    自由に線を描きます。
    ＋Shiftキー / 左右にドラッグすることでブラシサイズを変更できます。
  • 直線
    左ドラッグで始点と終点を指定し、直線を描画します。
    ＋Ctrlキー / 45度単位の線にします。
  • 四角枠
    左ドラッグで矩形範囲を指定し、四角形の枠を描画します。
    ＋Ctrlキー / 正方形にします。
    ＋Shiftキー / 最初に押した位置を中心とした四角形にします。
    ＋Altキー / 角丸にします（※ただし、「ぼかし」などの効果系ツールでは無効です）
  • 四角塗りつぶし
    塗りつぶしの四角形を描画します。
    操作は四角枠と同様です。
  • 円枠
    左ドラッグで円の枠を描画します。
    ＋Ctrlキー / 正円にします。
    ＋Shiftキー / 最初に押した位置を中心とした円にします。
  • 円塗りつぶし
    塗りつぶしの円を描画します。
    操作は四角枠と同様です。
  • 連続直線
    連続した直線を描画します。
    左クリックで最初の位置を指定、その後複数左クリックを行っていくと前回の位置とクリックした位置の間で直線を描画します。
    終了する場合は右クリックするかEnterかEscキーを押してください。
  • 集中線
    1点に集中した直線を描画します。
    左クリックで中心となる位置を指定した後、複数左クリックを行っていくと中心位置とクリックした位置の間で直線を描画します。
    終了する場合は右クリックするかEnterかEscキーを押してください。
  • ベジェ曲線
    始点・終点・2つの方向点の4点からベジェ曲線を描画します。
    (1) 直線の時と同じように左ドラッグで始点と終点を一度に指定します。
    (2) 1つ目の方向点を左クリックで指定します。
    (3) 2つ目の方向点を左クリックで指定します。
    (4) 曲線が描画されます。
    ※方向点指定時に右クリックまたはEnterかEscキーを押すと中断されます。
  • スプライン曲線
    指定した点をすべて通る曲線を描画します。
    左クリックで通過する点を複数指定し、右クリックまたはEnterかEscキーを押すと描画されます。
    Backspaceキーを押すと、一つ前に戻ることができます。

▼ コントロールウィンドウ

▼ボタンを押すと、下に隠れている部分を表示/非表示にできます。

  • ブラシプレビュー
    現在のブラシサイズの大きさが表示されます。
    反映されるのはブラシサイズだけなので、形状は変化しません。
  • サイズ
    ブラシの直径サイズをpx単位で指定します。
  • 濃度
    ブラシの濃度（もしくは各ツールの強さ）を指定します。
  • ブラシサイズ・濃度記録
    サイズと濃度指定の下にあるのは、頻繁に使うブラシサイズと濃度を記録して保存しておけるゾーンです。
    上の数値がブラシサイズ、下の数値が濃度です。
    この記録はツールごとに分かれているので、各ツール毎に7個ずつ記録できます。

    左クリックすると、そのサイズと濃度を呼び出し、現在のサイズと濃度にセットできます。
    右クリックまたはCtrl＋左クリックすると、現在のサイズと濃度をその位置に記録できます。

  • 色マスク
    色マスクを使うと、指定した色をマスクして描画できないようにしたり、逆マスクして指定色の上しか描画できないようにできます。
    指定した色と完全に一致する色しか対象にならないため、ブラシや水彩などで描画された色では対象になりません。
    マスク色は最大5色まで指定できます。
    色が追加されると、色プレビューの部分が色数分分割されて表示されます。

    [OFF] … 色マスクをOFFにします。
    [マスク] … 色マスクをONにし、指定されている色の上には描画できないようにします。
    [逆マスク] … 色マスクを逆マスクとしてONにし、指定されている色の上にしか描画できないようにします。

    [＋ADD] … 現在の描画色をマスク色に追加します。
    [−DEL] … 一番右端の色をマスク色から削除します。
    [CLR] … マスク色が複数ある場合、一番左端の色を残し1色だけにします。

    《 色プレビュー部分 》
    左クリック / 現在の描画色をその位置の色にセットできます。
    右クリック / メニューが表示されます。
    Ctrl＋左クリック / クリックした位置の色を削除します。
    Alt＋左クリック / クリックした位置の色を取得し、現在の描画色にします。

  • テクスチャ
    テクスチャをONにすると、指定されたパターンが描画時に適用されます。
    左クリックすると、テクスチャのON/OFFを切り替えられます。
    右クリックすると、テクスチャウィンドウを表示/非表示できます。

  • 描画色/背景色
    手前に表示されているのが描画する時に使用される色で、背面に表示されているのがグラデーション時やフィルタ時など2色が必要なときのみ使われる色です。
    それぞれ、左クリックすると手前と背面の状態が切り替わり描画色と背景色が入れ替わります。
  • 色バー
    RGBまたはHSVで描画色の色を調整できます。

  • 2色間のグラデーションバー
    左と右に2つの色を指定し、その2色の間でグラデーションを作成します。
    作成された色の部分で左クリック/ドラッグすると、その位置の色を取得して現在の描画色にできます。
    グラデーションの元になる2色の色を指定するには、現在の描画色に指定したい色をセットした上で左端と右端の部分でそれぞれ左クリックします。
  • HSVカラーマップ
    HSVカラーの色一覧が表示されます。
  • 定規
    描画タイプが自由線の時に、あらかじめ指定した角度の直線や指定した中心位置への集中線などになるように線を合わせることができます。
    機能をONにした上でキャンバス上で線を描くと、各タイプの形に合うような線が描けます。
    定規がOFF以外の状態になっている時にキャンバス上でAltキーを押しながら左クリックまたは左ドラッグすると、それぞれのタイプに応じた角度や位置などの情報を変更できます。

    [OFF]
    定規機能を使いません。通常の自由線を描きます。
    [平行線]
    指定した角度の平行線を描けるようにします。
    Altキーを押しながら左ドラッグで線を引き、角度を指定します。
    [集中線]
    指定した中心位置への集中線を描けるようにします。
    Altキーを押しながら左クリックで中心位置を指定します。
    [正円]
    指定した位置を中心とする正円を描けるようにします。
    Altキーを押しながら左クリックで中心位置を指定します。
    [楕円]
    指定した位置を中心とする楕円を描けるようにします。
    Altキーを押しながら左ドラッグで、最初にボタンを押した位置が中心位置、ドラッグして楕円の形を指定します。
    [格子線]
    指定した角度とその角度に対して90度の角度の直線を描けるようにします。
    Altキーを押しながら左ドラッグで線を引き、角度を指定します。

  • デスクトップ上の色取得
    スポイトのイメージがある部分で左ボタンを押し、そのままドラッグするとカーソル下にあるデスクトップ上の色を取得します。
    枠内に現在の位置の色が表示されるので、左ボタンを離すとその位置の色が描画色になります。

▼ レイヤウィンドウ

レイヤウィンドウに画像ファイルをD＆Dすると（複数ファイル可）、画像をレイヤとして追加できます。


＜レイヤ一覧部分＞

  • ■部分を左クリックするとレイヤの表示/非表示を切り替えられます。
  • 塗りつぶしマーク部分を左クリックすると 「塗潰判定元」 のチェックをON/OFFできます。
  • 左ダブルクリックするとレイヤ設定ウィンドウを表示できます。
  • ドラッグ＆ドロップするとレイヤの位置を移動できます。
  • 右クリックするとメニューが表示されます。
    （クリックした位置のレイヤをフィルタの対象や判定元として、現在のレイヤでそれぞれの処理を実行できます。
    処理自体は各フィルタで実行するのと同じですが、対象レイヤを指定して実行する手間を省くことができます）
  • Ctrl＋左クリックで、カレントレイヤ変更と 「カレントレイヤのみ表示」 を同時に行えます。

＜各レイヤ設定＞

  • 合成モード
    下のレイヤと合成する際の合成方法を指定できます。
  • 濃度
    レイヤの表示濃度を指定できます。
  • 塗潰判定元
    塗りつぶしツールで塗りつぶす時に、このチェックがあるレイヤを塗りつぶし範囲の範囲元に使用します。
    ※塗潰判定元にできるのは1つのレイヤのみです。
  • マスク
    [OFF] … マスクをOFFにします。
    [（透）透明色マスク] … レイヤで透明な部分は描画できないようにします。
    [（不）不透明色マスク] … レイヤで透明でない部分は描画できないようにします（透明な部分しか描画できません）
    [（α）アルファマスク] … アルファ値を現状のままで変化しないようにします。

＜メニュー＞

  • 追加 … 現在のレイヤの上に新規レイヤを追加します。
  • ファイルから追加 … ファイルから画像を読み込み新規レイヤとして追加します。
  • 複製 … 現在のレイヤをコピーしたものを新規作成します。
  • 削除 … 現在のレイヤを削除します。
  • 下のレイヤと結合 … 現在のレイヤと下のレイヤを結合して1つのレイヤにします（合成モードが通常同士でない場合正しく結合できません）
  • 表示レイヤをすべて結合 … 白色の背景とすべての表示レイヤを合成して1つのレイヤにします（結合後はすべて不透明になります）
  • 表示レイヤをすべて「通常」結合 … 合成モードは「通常」扱いですべての表示レイヤを結合します（アルファ値は維持されます）
  • 表示レイヤをすべて新規レイヤに「通常」結合 … 合成モードは「通常」扱いですべての表示レイヤを結合したものを新規レイヤに追加します（アルファ値は維持されます）
  • 設定 … レイヤの名前など、レイヤの設定を行います。
  • ファイルに保存 … 現在のレイヤを、アルファ値を維持した32ビットのPNGファイルに保存します。

▼ パレットウィンドウ

パレットは複数のタブで分けることができ、タブは10個まで作成できます。

色を左クリックすると、その色を取得して現在の描画色にできます。
Ctrl＋左クリックすると、現在の描画色をクリックした位置に登録できます。
右クリックの動作は「設定」で指定できます。

Altキーを押しながら左ドラッグすると、ボタンを押した位置と離した位置の色の間で、グラデーションを作成します。

＜メニュー＞

  • （タブ）追加 … タブを追加します。
  • （タブ）削除 … 現在のタブを削除します。
  • （タブ）パレット読込 … AzPainter用のパレットファイル（*.apl）を読み込みます。
  • （タブ）パレット保存 … 現在のタブのデータをAzPainter用のパレットファイルに保存します。
  • （タブ）画像からパレット読込 … 画像に使用されている色からパレットを読み込みます（フルカラーでも可）
  • （編集）パレット編集 … パレットを編集します。
  • （編集）HSV順にソート … 現在のタブのパレットをHSV順に並び替えます。
  • （編集）すべて白に … 現在のタブのパレットをすべて白色にします。
  • （編集）設定 … パレットウィンドウに関する設定を指定します。

＜パレット編集＞

  • 挿入 … 選択1の位置に白色を挿入します。
  • 削除 … 選択1の位置の色を削除します。
  • 選択間挿入 … 選択1と2の間に白色を挿入します。
  • 選択間削除 … 選択1と2の間の色を削除します。
  • 選択間グラデーション … 選択1と2の間でグラデーションを作成します。
  • RGB値
    選択1の位置の色をRGB値で指定します。
    R、G、Bの順でそれぞれ0〜255の値を指定し、各値との間には数字以外の適当な文字をはさみます。
    （無駄な空白などは含めないでください）
    「セット」ボタンで色がセットされます。

    （例） 0,128,255 → R 0 G 128 B 255 として扱われます。

▼ テクスチャウィンドウ

テクスチャの一覧が表示されています。
左クリックで選択を変更できます。
ウィンドウに画像ファイルをドラッグ＆ドロップするとそのイメージをテクスチャに登録できます（複数ファイル可）

＜テクスチャデータについて＞

1つのテクスチャデータは 1×1〜512×512 の可変サイズになっています。
1つの点は0〜255の濃淡値となっており、描画時には描画の濃度とテクスチャの濃度を掛け合わせた値で描画されます。

テクスチャを自作する場合は、テクスチャのイメージをBMP/PNG/GIF/JPEGファイルで用意し、テクスチャウィンドウから追加してください。
横・縦のサイズは最大で512pxまでです。それを超えるイメージの場合は512pxまでに切り取られます。

イメージは黒から白のグレイスケールにします（カラーのイメージの場合はグレイスケールに変換されます）
イメージ上での白色が透明（アルファ値=0）、黒色が不透明（アルファ値=255）となります。
黒色から白色になるほど透明になります。

▼ プレビューウィンドウ

プレビューウィンドウでは現在のイメージを特定の倍率で表示できます。
イメージの全体を確認しながら描画したり、拡大しながら描画しているときに等倍のイメージを確認したりできます。

▼ボタンをクリックするとメニューが表示されます。
左ドラッグすると、プレビューウィンドウ上のイメージの位置を移動できます。
右クリック/ドラッグすると、キャンバス上のイメージの中心位置をクリックした位置に移動できます。

▼ イメージビューウィンドウ

イメージビューウィンドウでは、ファイルから画像を読み込んで表示できます。
資料の絵などを表示しながら描くことが出来ます。
画像ファイルをウィンドウにドラッグ＆ドロップすると画像を読み込むことができます。

▼ボタンをクリックするとメニューが表示されます。
左ドラッグすると、イメージビューウィンドウ上のイメージの位置を移動できます。
右クリックすると、クリックした位置の色を現在の描画色にできます。

▼ フィルタ一覧ウィンドウ

フィルタの一覧が表示されているので、左クリックすることで手早くフィルタを実行できます。



▼ ファイル形式

  • APDファイル(*.apd)
    AzPainter専用の形式です。
    レイヤ状態などを維持したまま保存できます。
  • ADWファイル(*.adw)
    AzDrawing専用の形式です。
  • PSDファイル(*.psd)
    PhotoShopの形式です。
    読み込みは8bitのRGBのみです。CMYKなどのカラーモードの場合読み込めません。
  • BMPファイル(*.bmp)
    Windowsの標準形式です。
    無圧縮のためファイルサイズは大きくなります。
    256色以下の場合は8BIT、257色以上の場合は24BITで出力します。
  • PNGファイル(*.png)
    256色以下の場合は8BIT、257色以上の場合は24BITで出力します。
    アルファ付きPNGで保存すると、アルファ値を維持したまま全レイヤを1つの画像に結合したものが出力されます。
    （※レイヤの合成モードに「通常」以外のタイプが含まれていると、結合前のイメージと異なります）
  • JPEGファイル(*.jpg)
    ■ 保存時の設定
    [品質]
    1〜100で、値が低いほど画質が悪く（ファイルサイズが小さく）、値が高いほど画質が良く（ファイルサイズが大きく）なります。
    [ダウンサンプリング]
    若干画質を下げファイルサイズを重視します。
    チェックOFFの方がややはっきりとした画像になり、赤色の劣化を抑えられます。
    ON/OFFで画質の差はそれほど大きくありませんが、出来るだけ画質を良くしたい場合はOFFにしてください。
  • GIFファイル(*.gif)
    GIFは256色までなので、保存時に257色以上存在する場合は自動的に減色されます。

▼ フィルタ

選択範囲がある時はその範囲にだけ効果がかかります（一部全体に効果がかかるフィルタもあります）
なお、フィルタ処理時には色マスク機能・レイヤのマスク機能などは無効です。

[プレビューについて]
フィルタにはキャンバス上でプレビューできるものとフィルタ設定上でプレビューできるものがありますが、フィルタ設定上のプレビューは左ボタンのドラッグで表示位置を移動出来ます。


  • （カラー）描画色-背景色に着色
    現在のイメージの輝度（明るさ）を元に描画色から背景色に色が変化するように色を付けます。
    暗い色ほど描画色に、明るい色ほど背景色に近い色になります。
  • （色操作）色の置き換え
    現在の描画色と同じ色の部分を指定した色に置き換えます。
  • （アルファ操作）他のレイヤと操作
    現在のレイヤと他のレイヤを対象に、アルファ値の操作を行います。
    [対象レイヤで透明な部分は透明に] … 対象レイヤで透明な部分は現在のレイヤでも透明にします（不透明な部分はそのまま）
    [対象レイヤで不透明な部分は透明に] … 対象レイヤで透明でない部分は現在のレイヤで透明にします。
    [対象レイヤの値を掛ける] … 現在のレイヤと対象レイヤのアルファ値を掛けて255で割ります（乗算）
    [対象レイヤの色の輝度からコピー] … 対象レイヤの色の明るさをアルファ値としてコピーします。明るい色ほど不透明に、暗い色ほど透明になります。
    [対象レイヤの色の輝度から反転コピー] … 対象レイヤの色の明るさを反転した値をアルファ値としてコピーします。明るい色ほど透明に、暗い色ほど不透明になります。
  • （アルファ操作）アルファ値からグレイスケール作成
    アルファ値を元にグレイスケールイメージを作成します。
    アルファ値が0（透明）なら黒に、255（不透明）なら白になります。
  • （描画）不透明部分の縁取り
    現在の描画色と指定したブラシサイズで、不透明な部分の周り（外側）に点を描画します。
    実際に描画されるのは透明部分上だけなので、透明部分がないと意味がありません。
    「判定元レイヤ」は、指定したレイヤのイメージを判定元に使って現在のレイヤに縁取り出来ます。
  • （描画）不透明部分の内側を縁取り
    現在の描画色と指定したブラシサイズで、不透明な部分の内側に点を描画します。
  • 1ピクセル線の補正
    背景が透明で、1ピクセルのドットで描かれた線を細く綺麗にします。
    線が太くなっている部分は処理されません。
  • アンチエイリアシング
    ドットのイメージが滑らかになるようにアンチエイリアス処理を行います。
  • 輝度から線画抽出
    イメージの色の輝度を元にアルファ値を求め、色は黒で統一します。白い色ほど透明に、黒い色ほど不透明になります。
  • 縁取り

    • アンチエイリアス部分も考慮して不透明な部分の周りに縁取りを描画します。

▼ 環境設定

＜１＞

  • 1段階の拡大縮小幅
    表示倍率の1段階拡大・縮小を行う場合に、どれだけ倍率を増減させるかの値です。
    「〜100％」は、現在の倍率が100％以下の時に増減させる値です。
    「100％〜」は、現在の倍率が100％以上の時に増減させる値です。
    例えば、「100％〜」に100を指定すると、100、200、300…というように表示倍率が変化します。
  • アンドゥ回数
    アンドゥの最大回数を指定します。
    アンドゥデータは、サイズの大きいものはファイルに出力されます。
  • 情報バーの位置
    カーソル上のイメージ位置や表示倍率、イメージサイズが表示されているバーの表示位置を指定します。
  • 消しゴムデフォルトツール
    タブレットのサイドスイッチ動作やキー設定の「消しゴム・直前ツールの切替」時、それとスポイト時に透明色だった場合に消しゴムツールに切り替える時に、切り替える消しゴムツールのタイプを指定します。

＜２＞

  • 上書き保存時、確認する
    上書き保存時、確認メッセージを表示します。
  • アプリケーション終了時、保存確認する
    アプリケーション終了時にイメージに変更があり保存されていなかった場合、確認メッセージを表示します。
  • タイトルバーにはファイル名のみ表示する
    アプリケーションのタイトルバーには通常編集中のファイルのフルパスが表示されますが、フォルダ名を除いてファイル名だけ表示させます。
  • メインウィンドウを自動アクティブ
    他のウィンドウがアクティブな状態でキャンバス上でマウスカーソルが動いた場合、メインウィンドウを自動的にアクティブにします。
  • スポイト時、透明色の場合消しゴムツールに切替
    キャンバス上でスポイトを行ったときに透明色だった場合、ツールを消しゴムに変更します。
    （変更されるツールは「消しゴムデフォルトツール」で指定された消しゴムです）
    OFFの場合は、白色として扱います。
  • 描画中、ブラシサイズの円を表示
    自由線を描画している間、キャンバス上にブラシサイズの大きさの円を表示します。
    指定したサイズ以上の時だけ表示できます。
  • 消しゴムでの塗りつぶし時は判定元レイヤを無効にする
    塗りつぶしツールにて消しゴムで塗りつぶす場合、判定元レイヤを無効にして常にカレントレイヤを元に塗りつぶします。

＜タブレット＞

  • タブレット有効
    タブレットを有効にします。
    この設定は次回起動時に有効となります。
    OFFの場合は通常のマウス操作と同じ扱いで処理されます（筆圧無効）
  • マウス座標を使う
    タブレットドライバでマウスモードに設定している場合や、カーソル位置と描画位置がずれてどうしても直らない場合はチェックをONにしてください。
    タブレットの座標ではなくマウスカーソルの座標を使います。
    ずれは直りますが座標の精度が下がります。
  • テールスイッチ時のツール
    タブレットペンの消しゴム部分を使った時にどのツールに変更するかを指定します。
  • サイドスイッチの動作
    サイドスイッチ（タブレットペンのボタン）にそれぞれ特定の動作を割り当てることができます。
    この設定はキャンバス上で操作を行ったときのみ有効です。

▼ キー設定

キャンバス上で、指定したキーが押された時に各処理を実行できます。

  • ツール
    ツールを変更します。
  • 描画タイプ
    描画タイプを変更します。
  • コマンド
    各コマンドを実行します。
  • キー＋クリック/ドラッグ
    指定したキーを押しながら左クリックまたは左ドラッグを行うことで実行されます。

    [直線〜スプライン曲線] … キー＋ドラッグで、描画タイプを変えずに他の描画タイプで描画できます。
    [スポイト] … キー＋クリックで、スポイトを行います。
    [塗りつぶし] … キー＋クリックで、ツールの塗りつぶしを実行します（※塗りつぶしツールの設定が使われます）
  • その他
    [消しゴム・直前ツール切替] … 現在のツールが消しゴム以外の時は消しゴムに、消しゴムの時は直前のツールに切り替えます。消しゴムツールは環境設定の「消しゴムデフォルトツール」で指定されたツールが使用されます。
    [描画色・背景色切替] … 描画色/背景色の前面/背面を入れ替え、反対側を現在の描画色にします。

    原因はよくわかりませんが、一時的にマウスの操作として扱われる場合があるようです。
    intuos3タブレットの場合は、タブレットのプロパティでタブレットモードを標準から文字認識に変更したら直ったという報告があります。
    また、下の「タブレットで描画位置がずれます」の対処法と同じことをすることで直る場合があります。

  • タブレット使用時、描画位置がずれます
    タブレットドライバで「マウスモード」にしている場合は、正しく位置が取得できません。その場合は環境設定の「（タブレット）マウス座標を使う」をONにして使用してください。

    WACOMタブレットの場合は、タブレットの設定ファイルを削除して初期化することで直る場合があります。

    (1) ユーティリティがある場合
    「スタート」→「すべてのプログラム」→「ペンタブレット」→「ペンタブレット設定ファイルユーティリティ」を開き、「すべての設定ファイルを削除する」を実行する。

    (2) 設定ファイルを手動で削除する
    （WinXPの場合）C:\Windows\System32
    （Win2000の場合）C:\WinNT\System32
    （Win98/Meの場合）C:\Windows\System
    もしくは、C:\Documents and Settings\ユーザー名\Application Data\WTablet

    上記のフォルダ内にある「tablet.dat」または「wacom.dat」ファイルを削除した後再起動してみてください。
    これで直らない場合は、タブレットドライバを一度アンインストールして再起動した後、最新のドライバまたは安定版を再度インストールしてみてください。

  • 途中保存したり続きから描くには？
    AzPainter専用形式（*.apd）で保存すればレイヤの状態などを維持したまま保存できます。
    BMPなどの画像ファイルから読み込んで編集した場合は、「別名で保存」で「ファイルの種類」のところを「AzPainterファイル（*.apd）」に変更して保存すればAPD形式で保存できます。

  • ファイルの形式を変えて保存するには？
    「別名で保存」でファイルを指定する時に、「ファイルの種類」のところを「PNG（*.png）」など変更したい形式に変えます。
    そして保存すればOKです。

  • 画像を印刷するには？
    AzPainterには印刷機能はありませんので、画像をBMPやPNGで保存して、それを印刷機能のある他のソフトで読み込み、印刷してください。

  • 画像の解像度（dpi）を指定したい
    AzPainterでは、解像度は指定できません。
    通常、解像度は印刷する時などに必要な情報なので、普通に絵を描くだけならば解像度は必要ないためです。
    もし出力した画像の解像度を変更したい場合は、BMPやPNGなどの解像度を変更するソフトを使ってください。

  • 編集できる画像のサイズは？
    幅・高さが4500px以内であることが条件です。
    4501px以上のサイズは編集できません。

  • テキスト描画で縦書きが正しく描画されない
    縦書きにする時は先頭に「＠」のついたフォント名を指定してください。
    「＠」のついたフォント名は縦書き用です。
    逆に横書きの時に「＠」のついたフォント名を指定すると正しく横書きになりません。

  • テクスチャのデータが表示されない
    テクスチャファイル（*.apt）が存在しないか、ファイルにデータが存在しないものと思われます。
    再度AzPainterのアーカイブをダウンロードし、アーカイブ内にある「texture.apt」をAzPainterのフォルダにコピーしてください。
    ※AzPainterが起動している場合は先にソフトを終了させてください。

  • 表示倍率が大きい時に1ドットのグリッド線がずれる
    イメージサイズが大きくかつ表示倍率が大きい時は表示時の計算の誤差でグリッド線がずれる場合があります。
